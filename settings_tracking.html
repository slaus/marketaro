Settings - Tracking

State Setting Description
  • Inactive Click Tracking Overwrites every link to track every click in emails.

  • Overwrites every link to track clicks in e-mails.

  • Active Google Analytics Track your conversion rates and ROI with Google Analytics.

  • You will may add the info through the input field.

    Add

    Campaign source | Remove

  • Inactive Open Tracking Appends an invisible image to HTML emails to track emails that have been opened.

  • Tern On to enable open tracking.

  • Active Subscription Tracking Adds unsubscribe links to the bottom of the text and HTML emails. Future emails won't be delivered to unsubscribed users.

  • If you would like to place the unsubscribe text somewhere else in the email content, you can do so by placing an unsubscribe tag. You would need to define this tag here. If no tag is found, we will simply place the unsubscribe information at the bottom of the email.

    REPLACEMENT TAG

    UNSUBSCRIBE CONTENT - HTML BODY

    Please define what you would like your unsubscribe message (HTML version) to look like. You must include a <% %> tag to place the unsubscribe link.

    UNSUBSCRIBE CONTENT - TEXT BODY

    Please define what you would like your unsubscribe message (text version) to look like. You must include a <% %> tag to place the unsubscribe link.

    CUSTOM LANDING PAGE URL

    If you have your own landing page prepared, you can have us redirect the user there. Must be a valid URL.

    SENDGRID LANDING PAGE - HTML BODY

    Have SendGrid create a landing page for you.
